# Bella Vista 🍝

A modern and responsive restaurant landing page built using **HTML** and **CSS**.  
This project showcases a stylish homepage design for an Italian restaurant with a clean navigation bar, hero section, and elegant UI styling.

---

## 🚀 Features

- Responsive navigation bar
- Modern restaurant-style UI
- Hero section with call-to-action button
- Clean typography and layout
- Pure HTML & CSS project
- Beginner-friendly structure

---

## 🛠️ Technologies Used

- HTML5
- CSS3
